# Animasyon Dosyaları

Bu klasör, uygulama içinde kullanılan Lottie animasyon dosyalarını içerir.

## Gerekli Animasyon Dosyaları

### microphone.json
- **Açıklama**: Ana ekrandaki mikrofon animasyonu
- **Kullanım**: Kayıt öncesi durumda gösterilen animasyon
- **Boyut**: ~50-100KB
- **Kaynak**: LottieFiles.com

### baby_listening.json
- **Açıklama**: Onboarding ekranı - bebek dinleme animasyonu
- **Kullanım**: İlk ekranda uygulama özelliğini tanıtan animasyon
- **Boyut**: ~100-200KB

### offline.json
- **Açıklama**: Çevrimdışı çalışma özelliğini gösteren animasyon
- **Kullanım**: Onboarding ekranında çevrimdışı özellik tanıtımı
- **Boyut**: ~75-150KB

### security.json
- **Açıklama**: Güvenlik ve gizlilik animasyonu
- **Kullanım**: Onboarding ekranında güvenlik özelliği tanıtımı
- **Boyut**: ~80-160KB

### learning.json
- **Açıklama**: Federated learning özelliğini gösteren animasyon
- **Kullanım**: Onboarding ekranında öğrenme özelliği tanıtımı
- **Boyut**: ~90-180KB

## Animasyon Dosyalarını Ekleme

1. LottieFiles.com'dan uygun animasyonları indirin
2. JSON formatında olduklarından emin olun
3. Bu klasöre yukarıdaki isimlerle kaydedin
4. Boyut optimizasyonu için gereksiz katmanları kaldırın

## Alternatif Çözüm

Animasyon dosyaları yoksa, uygulama statik ikonlar kullanarak çalışmaya devam edecektir. `assets/icons/` klasöründe alternatif görseller bulunabilir.

## Lisans

Kullanılan animasyon dosyalarının lisanslarına uygun olarak kullanılması gerekmektedir. Ticari kullanım için gerekli lisansların alındığından emin olun.
