# Sakinleştirme Rehberi

Bu depo, "Rehber Ekranı"nın statik bir sürümünü içerir.

## Nasıl görüntülenir

- `index.html` dosyasını doğrudan tarayıcıda açın, veya
- Basit bir statik sunucu başlatın:
  - Python: `python3 -m http.server 8080`
  - Ardından: `http://localhost:8080/index.html`

## Sayfa düzeni

- Başlık: "Sakinleştirme Rehberi" (merkezde, mavi)
- Kartlar (300x150):
  - Açlık (🤱): kısa açıklama + video oynatıcı
  - Gaz (🚲): açıklama + video oynatıcı
  - Uykusuzluk (💤): açıklama + video oynatıcı
